== Description ==

The Maerquin WordPress theme based on the most powerful frameworks in the world:
    “_s” – by Automattic, the main people behind WordPress development
    Twitter Bootstrap – the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
    Font Awesome – a full suite of 675+ pictographic icons for easy scalable vector graphics on websites

This theme, is build for Maerquin only and not meant to distribute among other sites without the approval from the creator.

Page templates
* Front-page
* Search
* Page

Other features:
* Currently using Twitter Bootstrap v3.2.1
* Font Awesome integrated
* Mega menu
* Widgetized menu area
* Compatible with Contact Form 7

This theme will be an inactive project which we will only be updated when problems occur.

== Installation ==

1. Upload in the FTP
2. Click Activate to use your new theme right away.
